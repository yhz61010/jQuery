element

选择所有与指定标签名相同的元素。

  • 1.0 新增element

    element (String) 用于查找的元素。代表 DOM 节点的标签名。

使用该表达式时，会调用 JavaScript 的 getElementsByTagName() 函数返回适当的元素。

示例:

查找每一个 DIV 元素。

<!DOCTYPE html>
<html>
<head>
<style>
  div,span {
    width: 60px;
    height: 60px;
    float:left;
    padding: 10px;
    margin: 10px;
    background-color: #EEEEEE;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div>DIV1</div>

  <div>DIV2</div>
  <span>SPAN</span>

<script>

$("div").css("border","9px solid red");

</script>
</body>
</html>
