:header

选择所有的标题元素，比如 h1, h2, h3 等。

  • 1.2 新增:header

补充说明:

  • 由于 :header 是 jQuery 扩展出来的，它并不是 CSS 规范中的一部分。当使用 :header 时，并不会比使用原生的 DOM 方法 querySelectorAll() 性能好。为了达到高效的 :header 元素选择，请优先使用纯 CSS 选择器 .filter(":header") 。

示例:

给页面上所有的标题元素加上背景和文本颜色。

<!DOCTYPE html>
<html>
<head>
<style>
  body { font-size: 10px; font-family: Arial; } 
  h1, h2 { margin: 3px 0; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<h1>Header 1</h1>

  <p>Contents 1</p>
  <h2>Header 2</h2>
  <p>Contents 2</p>

<script>

$(":header").css({ background:'#CCC', color:'blue' });

</script>
</body>
</html>
