返回值:jQueryprevAll([selector])

查找匹配元素集合中每个元素前面的所有兄弟元素，可以提供一个可选的选择器。

  • 1.2 新增prevAll([selector])

    selector (Selector) 可选参数，用于匹配元素的选择器表达式字符串。

若一个 jQuery 对象代表了一个 DOM 元素集合，.prevAll() 方法允许我们在 DOM 树中查找集合中元素前面的所有兄弟元素，并根据匹配的元素创建一个新的 jQuery 对象。返回的元素是按照由近及远的顺序排列的。

该方法还可以接受一个可选的选择器表达式，该选择器表达式可以是任何可传给 $() 函数的选择器表达式。如果提供了选择器表达式，那么会先测试该元素是否满足匹配的选择器表达式。

例如，在页面中有如下一个简单的列表：

<ul>
   <li>list item 1</li>
   <li>list item 2</li>
   <li class="third-item">list item 3</li>
   <li>list item 4</li>
   <li>list item 5</li>
</ul>

如果我们从 item 3 开始，我们可以查找它前面所有的兄弟元素：

$('li.third-item').prevAll().css('background-color', 'red');

其结果是 item 1 和 item 2 的背景色变成了红色。由于我们没有提供选择器表达式，前面的这些兄弟元素很明确的成为了结果对象中的一部分。如果我们提供了选择器表达式，那么在包含在结果对象之前，会先测试该元素是否满足匹配的选择器表达式。

示例:

查找最后一个 div 前面的所有 div 元素，并给它们添加样式。

<!DOCTYPE html>
<html>
<head>
<style>

  div { width:70px; height:70px; background:#abc; 
        border:2px solid black; margin:10px; float:left; }
  div.before { border-color: red; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div></div>
  <div></div>
  <div></div>
  <div></div>

<script>

$("div:last").prevAll().addClass("before");

</script>
</body>
</html>
