[attribute~="value"]

选择那些属性值中含有给定单词的元素。单词以空格分隔。

  • 1.0 新增[attribute~="value"]

    attribute (String) 属性名。
    value (String) 使用单引号或双引号包裹的字符串形式的属性值。

这个选择器会用给定的字符串去匹配属性值中的每一个单词。这里“单词”的定义是以空格为界的字符串。只要给定的字符串等于属性中的任何一个单词，这个元素就会被选中。

示例:

查找所有属性中含有 'man' 这个单词的文本框，并且修改其文本值。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<input name="man-news" />

  <input name="milk man" />
  <input name="letterman2" />
  <input name="newmilk" />

<script>

$('input[name~="man"]').val('mr. man is in it!');

</script>
</body>
</html>
