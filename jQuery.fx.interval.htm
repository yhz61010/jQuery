返回值:NumberjQuery.fx.interval

动画的显示帧率（以毫秒为单位）。

  • 1.4.3 新增jQuery.fx.interval

该属性可以设置动画每秒显示的帧数。该属性的默认值是 13 毫秒。该属性值越小，在速度较快的浏览器中（例如，Chrome），动画执行的越流畅，但是会影响程序的性能并且占用更多的 CPU 资源。

由于在 jQuery 中该属性是全局有效的，因此当该值发生变化时，要想使该值生效，应该停止所有的动画。

注意: jQuery.fx.interval 目前对支持 requestAnimationFrame 属性的浏览器不起作用，例如 Google Chrome 11。此问题会在今后发布的版本中得以解决。

示例:

让所有动画以更少的帧数执行。

<!DOCTYPE html>
<html>
<head>
<style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
<script src="jquery.min.js"></script>
</head>
<body>

<p><input type="button" value="Run"/></p>
<div></div>

<script>


jQuery.fx.interval = 100;

$("input").click(function(){
  $("div").toggle( 3000 );
});
  

</script>
</body>
</html>
