返回值:jQuerysubmit(handler(eventObject))

选择所有提交按钮，即 type 为 submit 的元素。

  • 1.0 新增submit(handler(eventObject))

    handler(eventObject) (Function)

  • 1.4.3 新增submit([eventData], handler(eventObject))

    eventData (Object) 可选参数，
    handler(eventObject) (Function)

  • 1.0 新增submit()

:submit 选择器主要用于 button 或者 input 元素。注意，某些浏览器默认把没有设置 type 的 <button> 元素看做 type="default"，而另一些浏览器(比如IE)则并非如此。

补充说明:

  • 由于 :submit 是 jQuery 扩展出来的，它并不是 CSS 规范中的一部分。当使用 :submit 时，并不会比使用原生的 DOM 方法 querySelectorAll() 性能好。为了在主流浏览器中得到更好的性能，请使用 [type="submit"] 方法来代替。

示例:

查找所有作为 td 元素后代的提交按钮。

<!DOCTYPE html>
<html>
<head>
<style>

  p { margin:0; color:blue; }
  div,p { margin-left:10px; }
  span { color:red; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<p>Type 'correct' to validate.</p>
  <form action="javascript:alert('success!');">
    <div>
      <input type="text" />

      <input type="submit" />
    </div>
  </form>
  <span></span>

<script>



    $("form").submit(function() {
      if ($("input:first").val() == "correct") {
        $("span").text("Validated...").show();
        return true;
      }
      $("span").text("Not valid!").show().fadeOut(1000);
      return false;
    });


</script>
</body>
</html>
