返回值:booleanjQuery.isFunction(obj)

Determine if the argument passed is a Javascript function object.

  • 1.2 新增jQuery.isFunction(obj)

    obj (Object) Object to test whether or not it is a function.

注意: As of jQuery 1.3, functions provided by the browser like alert() and DOM element methods like getAttribute() are not guaranteed to be detected as functions in browsers such as Internet Explorer.

Test a few parameter examples.
