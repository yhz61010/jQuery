返回值:Deferreddeferred.resolve(args)

Resolve a Deferred object and call any doneCallbacks with the given args.

  • 1.5 新增deferred.resolve(args)

    args (Object) Optional arguments that are passed to the doneCallbacks.

When the Deferred is resolved, any doneCallbacks added by deferred.then or deferred.done are called. Callbacks are executed in the order they were added. Each callback is passed the args from the .resolve(). Any doneCallbacks added after the Deferred enters the resolved state are executed immediately when they are added, using the arguments that were passed to the .resolve() call.了解更多内容，请参见如下文档 延迟对象.
