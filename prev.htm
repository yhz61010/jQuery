返回值:jQueryprev([selector])

查找匹配元素集合中每个元素前面的直接兄弟元素，可以提供一个可选的选择器。

  • 1.0 新增prev([selector])

    selector (Selector) 可选参数，用于匹配元素的选择器表达式字符串。

若一个 jQuery 对象代表了一个 DOM 元素集合，.prev() 方法允许我们在 DOM 树中查找集合中元素前面的兄弟元素，并根据匹配的元素创建一个新的 jQuery 对象。

该方法还可以接受一个选择器表达式，该选择器表达式可以是任何可传给 $() 函数的选择器表达式。如果提供了选择器，那么只有前面的直接兄弟元素满足选择器时，才会被过滤出来。

例如，在页面中有如下一个简单的列表：

<ul>
   <li>list item 1</li>
   <li>list item 2</li>
   <li class="third-item">list item 3</li>
   <li>list item 4</li>
   <li>list item 5</li>
</ul>

若要选择 item 3 前面的直接兄弟元素，可以像下面这样:

$('li.third-item').prev().css('background-color', 'red');

其结果是 item 2 的背景色变成了红色。由于我们没有提供选择器表达式，所以前面的直接兄弟元素很明确的成为了结果对象中的一部分。如果我们提供了选择器表达式，那么在包含在结果对象之前，会先测试该元素是否满足匹配的选择器表达式。

如果前面的直接兄弟元素不存在，或者它不满足指定的选择器，那么会返回空的 jQuery 对象。

若要选择前面所有的兄弟元素，而不是只选择邻近的在前面的兄弟元素的话，请使用 .prevAll() 方法。

示例:

查找每个 div 前面的直接兄弟元素。

<!DOCTYPE html>
<html>
<head>
<style>
  div { width:40px; height:40px; margin:10px;
        float:left; border:2px blue solid; 
        padding:2px; }
  span { font-size:14px; }
  p { clear:left; margin:10px; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div></div>
  <div></div>
  <div><span>has child</span></div>

  <div></div>
  <div></div>
  <div></div>
  <div id="start"></div>

  <div></div>
  <p><button>Go to Prev</button></p>

<script>


    var $curr = $("#start");
    $curr.css("background", "#f99");
    $("button").click(function () {
      $curr = $curr.prev();
      $("div").css("background", "");
      $curr.css("background", "#f99");
    });



</script>
</body>
</html>

示例:

查找每个段落前面的直接兄弟元素，要求它含有 "selected" 样式。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<div><span>Hello</span></div>

  <p class="selected">Hello Again</p>
  <p>And Again</p>

<script>

$("p").prev(".selected").css("background", "yellow");

</script>
</body>
</html>
