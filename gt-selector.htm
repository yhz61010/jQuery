:gt(index)

选择在匹配的集合中，索引值大于 index 的元素。

  • 1.0 新增:gt(index)

    index (Number) 从 0 开始计数的索引值。

索引值相关的选择器

索引值相关的选择器 (:eq(), :lt(), :gt(), :even, :odd)，会在先前提供的选择器筛选出的元素基础上再进行筛选。进一步筛选的依据就是这个元素在原先匹配集合中的顺序。举例来说，如果一开始通过 class 选择器 (.myclass) 选中了 4 个元素，然后这四个元素的索引值会被分配为 0 到 3，之后就可以用这种索引值选择器来进一步筛选了。

注意，由于 JavaScript 数组的索引值是从 0 开始计数的，所以这些选择器也是以 0 开始计数的。例如，$('.myclass:gt(1)') 就能选中文档中 class 是 myclass 的第二个元素以后的那些元素，而不是第一个元素之后的那些元素。相反，:nth-child(n) 中的 n 是从 1 开始计数的，因为 CSS 规范里是这么设定的。

补充说明:

  • 由于 :gt() 是 jQuery 扩展出来的，它并不是 CSS 规范中的一部分。当使用 :gt() 时，并不会比使用原生的 DOM 方法 querySelectorAll() 性能好。为了在主流浏览器中得到更好的性能，请使用 $("your-pure-css-selector").slice(index) 方法来代替。

示例:

查找索引值大于等于 5 的 td。注意，索引值是从 0 开始计数的。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<table border="1">

    <tr><td>TD #0</td><td>TD #1</td><td>TD #2</td></tr>
    <tr><td>TD #3</td><td>TD #4</td><td>TD #5</td></tr>

    <tr><td>TD #6</td><td>TD #7</td><td>TD #8</td></tr>
  </table>

<script>

$("td:gt(4)").css("text-decoration", "line-through");

</script>
</body>
</html>
