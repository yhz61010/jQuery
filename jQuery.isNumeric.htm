返回值:BooleanjQuery.isNumeric(value)

判断传入的参数是否为数字。

  • 1.7 新增jQuery.isNumeric(value)

    value (Object) 将要被检查的值。

$.isNumeric() 方法会检查传入的参数是否为数字。如果是，则返回 true。否则返回 false。传入的参数可以是任何类型。

示例:

以下是使用 $.isNumeric 检测各种输入类型的例子。

jQuery 代码:

$.isNumeric("-10");  // true
$.isNumeric(16);     // true
$.isNumeric(0xFF);   // true
$.isNumeric("0xFF"); // true
$.isNumeric("8e5");  // true (exponential notation string)
$.isNumeric(3.1415); // true
$.isNumeric(+10);    // true
$.isNumeric(0144);   // true (octal integer literal)
$.isNumeric("");     // false
$.isNumeric({});     // false (empty object)
$.isNumeric(NaN);    // false
$.isNumeric(null);   // false
$.isNumeric(true);   // false
$.isNumeric(Infinity); // false
$.isNumeric(undefined); // false
