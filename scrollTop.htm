返回值:IntegerscrollTop()

得到第一个匹配元素的垂直滚动条的位置。

  • 1.2.6 新增scrollTop()

垂直滚动条的位置与元素上可滚动区域隐藏的像素数是相同的。（原文如下：The vertical scroll position is the same as the number of pixels that are hidden from view above the scrollable area.）如果滚动条处于最上边，或者元素是不可滚动的，那么该值是 0。

示例:

取得段落的 scrollTop 值。

<!DOCTYPE html>
<html>
<head>
<style>
  p { margin:10px;padding:5px;border:2px solid #666; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<p>Hello</p><p></p>

<script>

var p = $("p:first");
$("p:last").text( "scrollTop:" + p.scrollTop() );



</script>
</body>
</html>

返回值:jQueryscrollTop(value)

为每个匹配的元素设置垂直滚动条的位置。

  • 1.2.6 新增scrollTop(value)

    value (Number) 代表新的垂直滚动条位置的整数。

垂直滚动条的位置与元素上可滚动区域隐藏的像素数是相同的。（原文如下：The vertical scroll position is the same as the number of pixels that are hidden from view above the scrollable area.）该方法会为每个匹配的元素设置 scrollTop。

示例:

为 div 设置 scrollTop。

<!DOCTYPE html>
<html>
<head>
<style>
div.demo {
background:#CCCCCC none repeat scroll 0 0;
border:3px solid #666666;
margin:5px;
padding:5px;
position:relative;
width:200px;
height:100px;
overflow:auto;
}
  p { margin:10px;padding:5px;border:2px solid #666;width:1000px;height:1000px; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div class="demo"><h1>lalala</h1><p>Hello</p></div>

<script>

$("div.demo").scrollTop(300);


</script>
</body>
</html>
