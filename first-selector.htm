:first

选择第一个匹配到的元素。

  • 1.0 新增:first

:first 伪类选择器等价于 :eq(0)。也等价于 :lt(1)，因为这也只能匹配到第一个元素。 :first-child 则可以匹配到多个元素：即每个父元素中的第一个元素。

补充说明:

  • 由于 :first 是 jQuery 扩展出来的，它并不是 CSS 规范中的一部分。当使用 :first 时，并不会比使用原生的 DOM 方法 querySelectorAll() 性能好。为了达到高效的 :first 元素选择，请优先使用纯 CSS 选择器 .filter(":first") 。
  • 被选中元素的顺序和它们在文档中出现的顺序是一致的。

示例:

查找表格中的第一行。

<!DOCTYPE html>
<html>
<head>
<style>

  td { color:blue; font-weight:bold; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<table>
    <tr><td>Row 1</td></tr>
    <tr><td>Row 2</td></tr>

    <tr><td>Row 3</td></tr>
  </table>

<script>

$("tr:first").css("font-style", "italic");

</script>
</body>
</html>
