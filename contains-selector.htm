:contains(text)

选择含有特定文本的元素。

  • 1.1.4 新增:contains(text)

    text (String) 用于查找的文本，大小写敏感。

匹配的文本可以直接出现在选中的元素之中，或者出现在这个元素的后代元素中。作为一个属性选择器，:contains() 的括号中的文本可以直接书写字符串，也可以用引号包裹一下。只有完全匹配（包括大小写）的文本才会被选中。

示例:

查找所有含有 "John" 的 div 并且添加下划线。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>


<div>John Resig</div>

<div>George Martin</div>
<div>Malcom John Sinclair</div>
<div>J. Ohn</div>
      
    

<script>


$("div:contains('John')").css("text-decoration", "underline");
    

</script>
</body>
</html>
