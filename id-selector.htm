#id

根据指定的 id 属性，选择单一元素。

  • 1.0 新增#id

    id (String) 用于查找的 ID。即元素的 id 属性。

对于 id 选择器，jQuery 会使用极其高效的 JavaScript 函数 document.getElementById() 来实现。当其它选择器与 id 选择器一起使用时，例如 h2#pageTitle，jQuery 在确定匹配的元素之前会执行额外的验证。

无论任何时候，要时刻记得，身为一名开发者，你的时间通常才是最宝贵的资源。不要一味的关注如何优先选择器的速度，除非有明确的迹象表明，必须要改善它的性能。

在文档中，每一个 id 只能出现一次。如果多个元素使用了相同的 ID，那么 id 选择器只能匹配到 DOM 中的第一个满足条件的 id 元素。我们不应该依赖此种行为。在文档中，多个元素使用同一个 id 是无效的。

若 id 中包含句号或冒号之类的符号时，应该对这些符号使用反斜杠进行编码。

示例:

查找 id 为 "myDiv" 的元素。

<!DOCTYPE html>
<html>
<head>
<style>
  div {
    width: 90px;
    height: 90px;
    float:left;
    padding: 5px;
    margin: 5px;
    background-color: #EEEEEE;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div id="notMe"><p>id="notMe"</p></div>

  <div id="myDiv">id="myDiv"</div>

<script>

$("#myDiv").css("border","3px solid red");

</script>
</body>
</html>

示例:

查找 id 为 "myID.entry[1]" 的元素。看看是如何使用反斜杠对特殊字符进行编码的。

<!DOCTYPE html>
<html>
<head>
<style>
  div {
    width: 300px;
    float:left;
    padding: 2px;
    margin: 3px;
    background-color: #EEEEEE;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div id="myID.entry[0]">id="myID.entry[0]"</div>

  <div id="myID.entry[1]">id="myID.entry[1]"</div>
  <div id="myID.entry[2]">id="myID.entry[2]"</div>

<script>

$("#myID\\.entry\\[1\\]").css("border","3px solid red");

</script>
</body>
</html>
