.class

Selects all elements with the given class.

  • 1.0 新增.class

    class (String) A class to search for. An element can have multiple classes; only one of them must match.

For class selectors, jQuery uses JavaScript's native getElementsByClassName() function if the browser supports it.

示例:

Finds the element with the class "myClass".

<!DOCTYPE html>
<html>
<head>
<style>
  div,span {
    width: 100px;
    height: 40px;
    float:left;
    padding: 10px;
    margin: 10px;
    background-color: #EEEEEE;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div class="notMe">div class="notMe"</div>

  <div class="myClass">div class="myClass"</div>
  <span class="myClass">span class="myClass"</span>

<script>

$(".myClass").css("border","3px solid red");

</script>
</body>
</html>

示例:

Finds the element with both "myclass" and "otherclass" classes.

<!DOCTYPE html>
<html>
<head>
<style>
  div,span {
    width: 100px;
    height: 40px;
    float:left;
    padding: 10px;
    margin: 10px;
    background-color: #EEEEEE;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<div class="myclass">div class="notMe"</div>

  <div class="myclass otherclass">div class="myClass"</div>
  <span class="myclass otherclass">span class="myClass"</span>

<script>

$(".myclass.otherclass").css("border","13px solid red");

</script>
</body>
</html>
