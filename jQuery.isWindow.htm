返回值:booleanjQuery.isWindow(obj)

判断传入的参数是否为 window 对象。

  • 1.4.3 新增jQuery.isWindow(obj)

    obj (Object) 将要被检查的对象。用于判断该对象是否为 window。

该方法在 jQuery 的很多地方都有使用，用来判断是否操作于浏览器的窗口中(例如，当前窗口或 iframe)。

示例:

判断传入的参数是否为 window。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

Is 'window' a window? <b></b>

<script>

$("b").append( "" + $.isWindow(window) );

</script>
</body>
</html>
