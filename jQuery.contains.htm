返回值:BooleanjQuery.contains(container, contained)

检测一个DOM节点是否包含另一个DOM节点。

  • 1.4 新增jQuery.contains(container, contained)

    container (Element) 可能包含其它元素的DOM元素。
    contained (Element) 可能被其它元素包含的DOM节点。

注意： 第一个元素必须是 DOM 元素，不能是 jQuery 对象或纯 JavaScript 对象。

示例:

检测一个元素是否在另一个元素之内。 不支持文档和注释节点。

jQuery 代码:
jQuery.contains(document.documentElement, document.body); // true
jQuery.contains(document.body, document.documentElement); // false
